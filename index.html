Programming Glossary
(for Non-Programmers)

Program: A set of instructions to make a computer do something, usually a specific thing like a calculator or a game.
Example: I like downloading new programs online just to try new things on my computer

Code: A set of instructions to make a computer do something, usually a specific thing like a calculator or a game. Code usually refers to the text of the intructions
Example: Bob looked thru the code to see if there were any errors before he sent it to a friend

Programming: Programming is the action of creating instructions for a computer to make it do something.
Example: Jane did some programming on her laptop to make a video game

HTML: HTML is one of the programming languages that is used to make websites. It is used to put all of the text, images, etc on the page. All websites use HTML.
Example: I learned how to use HTML in my computer science class yesterday

CSS: CSS is a programming language used to change the color, layout, sizes, etc of a webpage. Most websites use CSS.
Example: Changing the background color of a website is very easy with CSS

JavaScript: JavaScript is a computer language that is used to make a website interactive, like when you click on a button and something shows up on the page. Most websites use Javascript.
Example: Javascript is the most popular computer language used today

GitHub: GitHub is a website used by mainly developers to collaborate on coding. It enables multiple developers to work on a program at the same time easily.
Example: John logs into GitHub pretty much every day

Git: Git is a program used to make it easy to store different versions of a program. It is mainly used by programmers when working on improving their programs. If one tries to add new features to their program and it breaks the program, one can easily "go back" to a previous version of their program that worked using git.
Example: They recomend all programmers use git if they are working on a big program

Terminal: Terminal referms to a text-based way of instructing a computer to do something. It is mainly used by programmers when working with a computer as using a terminal gives one alot more control over the computer.
Example: She learned a few terminal commands and tried them on her laptop
